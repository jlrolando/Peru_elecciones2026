# Peru_elecciones2026
Pipeline en R para las elecciones de Perú 2026. Extrae datos de la API de ONPE nivel distrital y extranjero. Incluye lógica de extrapolación robusta: imputación por promedios de provincia/departamento, manejo de ubigeos y proyección de segunda vuelta. Ideal para análisis electoral técnico con manejo de rate-limiting.
